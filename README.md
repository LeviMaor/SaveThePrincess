<h1>Save The Princess</h1>
<h2>A short fighting game implemented using Unity and C#</h2>
<h3>Controllers:</h3>
<h4>Movement- Arrow keys</h4>
<h4>Dash- 'Z'</h4>
<h4>Sword attack- 'X'</h4>
<h4>Fireball attack- 'C'</h4>
<h4>Jump- Space</h4>
